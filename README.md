# Plancher Expert

**Calculez vos coûts de plancher en un instant !**  

Plancher Expert est un logiciel C# qui automatise le calcul des **matériaux et de la main-d'œuvre** pour tout projet de couvre-planchers. Entrez la surface et le type de plancher, et obtenez **immédiatement une estimation précise**.

# Fonctionnalités clés

- Calcul automatique **coûts matériaux + main-d'œuvre**  
- Interface simple pour entrer la **surface et le type de plancher**  
- Résultat **instantané et fiable**  
- Code **propre et structuré** démontrant les bonnes pratiques en C#

# Comment ça marche

1. Entrer la **surface à couvrir** (en pieds carrés)  
2. Choisir le **type de plancher** (bois, vinyle, stratifié…)  
3. Obtenir **coût total automatique**  

# Technologies

- C#  
- Visual Studio / .NET

# Instructions pour exécuter

1. Ouvrir le projet `Plancher Expert.csproj` dans **Visual Studio**.  
2. Appuyer sur **F5** pour lancer le logiciel.  
3. Suivre les instructions à l’écran et entrer la surface + type de plancher pour obtenir le calcul instantané.
